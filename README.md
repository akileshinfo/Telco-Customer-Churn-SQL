# Telco Customer Churn Analysis using SQL

## Project Overview

This project analyzes a telecom customer churn dataset using MySQL to identify customer behavior and factors contributing to churn.

## Tools Used

- MySQL
- SQL
- Kaggle Dataset

## Dataset

- Total Customers: 7043
- Features: 21

## SQL Concepts

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- CASE
- Subquery

## Business Questions

- Total Customers
- Churn Rate
- Gender-wise Churn
- Contract Analysis
- Internet Service Analysis
- Payment Method Analysis
- Revenue Analysis

## Key Insights

- Churn Rate: 26.54%
- Month-to-month customers churn the most.
- Electronic check customers have the highest churn.
- Fiber optic users show higher churn than DSL users.
